		[1, 4, 3, 0, 4, 5, 4]  //  => 2+2+2 = 6

		Задача 5.
		Напишите функцию поиска суммы максимального и минимального значения в массиве (не используя Math.max и Math.min)
		Пример:
		findMax([1, 12, 5, -6, 7, -8, -22, 3, -49, 0]) // => returns   -49 + 12 = -37

		createPhoneNumber([1, 2, 3, 4, 5, 6, 7, 8, 9, 0]) // => returns "(123) 456-7890"
